# Surprise Girlfriend Website - Option 2

Versi ini dibuat supaya lebih gampang di-deploy:
- tanpa `@/components/ui/*`
- tanpa shadcn/ui
- cukup React + Vite + framer-motion + lucide-react

## Jalankan lokal

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy ke Vercel

1. Upload folder ini ke GitHub
2. Import repo ke Vercel
3. Framework preset: Vite
4. Build command: `npm run build`
5. Output directory: `dist`

## File utama untuk edit
- `src/App.jsx`
